# FRAMEWORK DE CONVERSIÓN VIRAL (NEAPS + AIDA)
* **N (Núcleo del Dolor)**: Iniciar con el gancho agresivo de alta fricción (Ej: Multas, Cierres, Pérdida de capital).
* **E (Entorno Regulatorio)**: Explicar la realidad venezolana cambiante.
* **A (Atención Visual)**: Cortes visuales dinámicos cada 2-3 segundos en formato 9:16. Subtítulos fijos y tipografía Bold centrada.
* **P (Propuesta de Valor)**: Cómo el método automatizado de CEINCA resuelve la fricción.
* **S (Solución/Call to Benefit)**: Llamado a la acción de fricción cero utilizando palabras clave exactas para activar automatizaciones (Ej: COMENTA RECONVERSION).
